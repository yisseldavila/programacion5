Uso de Colores en HTML

Este párrafo demuestra cómo aplicar colores al texto usando diferentes métodos.

Este texto usa color en formato HEX (#FF5733), que es una forma común en diseño web.

Este texto usa color en formato RGB (60, 179, 113), útil para definir mezclas de color exactas.

Este texto usa color en formato HSL (240, 100%, 50%), que permite controlar tono, saturación y luminosidad.
